# PresençaLog — Protótipo inicial

Primeira versão navegável criada a partir da planilha `SSC8 - ESCALA DHL 5X2.xlsx`.

## Incluído
- 147 posições de escala importadas
- Login demonstrativo por ID Groot
- Consulta individual da escala
- Calendário de trabalho e folgas
- Painel do gestor
- Filtros por data, turno, cargo, nome e ID Groot
- Identificação das posições de REPOSIÇÃO

## Importante
Esta é uma versão de protótipo:
- não possui senha real;
- não possui banco de dados online;
- não permite editar a escala;
- ainda não deve ser usada como sistema oficial.

A próxima etapa é enviar estes arquivos ao repositório e publicar no GitHub Pages.
